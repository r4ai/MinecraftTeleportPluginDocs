# Items

## TeleportBerry

TeleportBerry is a berry that teleports the player when eaten.

### Usage

1. Holding the berry in your hand, <Kbd>shift</Kbd> + <Kbd>left-click</Kbd> to set the teleport location.
2. Then, eating the berry, you will be teleported to the remembered location.

<YouTube id="jFmXVjLM1XQ" title="TeleportBerry usage" />

### Recipe

<Recipe
  patterns={[' B ', 'BGB', ' B ']}
  keys={{
    B: { item: 'minecraft:sweet_berries' },
    G: { item: 'minecraft:gold_nugget' },
  }}
  result={{
    name: 'TeleportBerry',
    item: 'minecraft:sweet_berries',
  }}
/>

```json
{
  "pattern": [" B ", "BGB", " B "],
  "key": {
    "B": {
      "item": "minecraft:sweet_berries"
    },
    "G": {
      "item": "minecraft:gold_nugget"
    }
  }
}
```

> Defined in [TeleportBerry.kt](https://github.com/r4ai/MinecraftTeleportPlugin/blob/main/src/main/kotlin/com/github/r4ai/items/TeleportBerry.kt)

## HomeTeleportFeather

HomeTeleportFeather is a feather that teleports the player to the home location when <Kbd>right-click</Kbd>.

### Usage

1. Holding the feather in your hand, <Kbd>right-click</Kbd> to teleport to the home location.  
   Teleporting to home takes 1 second.

<YouTube id="1fxr9g9fThM" title="HomeTeleportFeather usage" />

### Recipe

<Recipe
  patterns={[' F ', ' D ', ' F ']}
  keys={{
    F: { item: 'minecraft:feather' },
    D: { item: 'minecraft:diamond' },
  }}
  result={{
    name: 'HomeTeleportFeather',
    item: 'minecraft:feather',
  }}
/>

> Defined in [HomeTeleportFeather.kt](https://github.com/r4ai/MinecraftTeleportPlugin/blob/main/src/main/kotlin/com/github/r4ai/items/HomeTeleportFeather.kt)

## MagicMoveWand (Experimental)

MagicMoveWand is a wand that pushes the player forward when <Kbd>right-click</Kbd>.

<Callout type="warning">
  This item is under development and is broken. Also, this item may be removed in the future.
</Callout>

### Usage

1. Holding the wand in your hand, <Kbd>right-click</Kbd> to go forward.
